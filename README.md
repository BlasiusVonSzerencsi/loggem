# Loggem

Simple log wrapper with different formatters. Minimal Rails/ActiveRecord/Sequel support, so far

## Installation

Add this line to your application's Gemfile:

    gem 'loggem'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install loggem
